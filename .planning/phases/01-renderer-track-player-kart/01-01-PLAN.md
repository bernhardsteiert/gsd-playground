    1. Grab canvas: `document.querySelector<HTMLCanvasElement>('#c')!`

  <action>
    Create `src/game/GameLoop.ts`:

    Constants at module top: `FIXED_DT = 1 / 60`, `MAX_DELTA = 0.1`.
    Module-level state: `let lastTimestamp = 0`, `let accumulator = 0`, `let _rafId = 0`.
    Export an `updateFn` type: `type UpdateFn = (fixedDt: number) => void`.

    Export `class GameLoop` with static method `start(renderer, scene, camera, onFixedUpdate?: UpdateFn)`:
    - Attach `document.addEventListener('visibilitychange', ...)`: on hidden do nothing (rAF pauses automatically); on visible set `lastTimestamp = 0` (forces MAX_DELTA cap on next frame per D-03).
    - Inner `loop(timestamp: number)` function:
      - Compute `rawDt = (timestamp - lastTimestamp) / 1000`; if `lastTimestamp === 0` set `rawDt = 0`
      - `lastTimestamp = timestamp`
      - `const dt = Math.min(rawDt, MAX_DELTA)` (D-02 cap)
      - `accumulator += dt`
      - `while (accumulator >= FIXED_DT) { onFixedUpdate?.(FIXED_DT); accumulator -= FIXED_DT; }`
      - `renderer.render(scene, camera)` (camera system update happens via onFixedUpdate for now)
      - FPS display (DEV only): `if (import.meta.env.DEV) { document.title = \`\${Math.round(1 / (dt || 0.016))} FPS | \${renderer.info.render.calls} draws\`; }`
      - `_rafId = requestAnimationFrame(loop)`
    - Bootstrap: `_rafId = requestAnimationFrame((ts) => { lastTimestamp = ts; _rafId = requestAnimationFrame(loop); })`

    Export `GameLoop.stop()` static method: `cancelAnimationFrame(_rafId)`.

    Create `src/utils/contextLoss.ts`:

    Export `function setupContextLossHandlers(canvas: HTMLCanvasElement): void`.
    Inside: attach `canvas.addEventListener('webglcontextlost', (e) => { e.preventDefault(); showOverlay(); })` and `canvas.addEventListener('webglcontextrestored', () => hideOverlay())`.

    `function showOverlay()`: create a `div` with `id='ctx-lost'`, inline style `position:fixed; inset:0; background:#000c; color:#fff; font:bold 28px sans-serif; display:flex; align-items:center; justify-content:center; z-index:9999`. Text content: `'Tap to resume'`. On `click`: `location.reload()`. Append to `document.body`.

    `function hideOverlay()`: `document.getElementById('ctx-lost')?.remove()`.
  </action>
